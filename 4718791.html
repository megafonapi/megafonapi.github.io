Мегафон - Телеком API : МАТЕРИАЛЫ ДЛЯ ХАКАТОНОВ

Created by Sergei Naumov, last modified on Dec 03, 2018

Для работы на хакатоне "47 часов" разработчикам предоставляется набор тестовых номеров, API-подписок  (6 штук). Для получения подписки необходимо:

  1. зайти на http://83.149.26.165/v0/devportal и зарегистрироваться, указав электронный адрес
  2. после входа на портал выбрать номер телефона подписки, с которым вы будете работать. Подписку можно освобождать от учетной записи и привязывать снова. В терминах МегаФон.API это anum. Просьба относится к этому ресурсу бережно, уважайте своих коллег. В момент привязки также будет сгенерирован JWT-токен для аутентификации на платформе:
  3. в дальнейшем необходимо использовать JWT-токен в своем коде. Если вы предполагаете писать JS-код для работы в браузере, где нет возможности добавлять HTTP-заголовки внутрь WebSocket'ов, используйте Basic-аутентификацию с логином и паролем. Логином в данном случае будет API-подписка, а паролем - пароль от зарегистрированной на портале учетной записи 
  4. документацию читать на http://83.149.26.165/v0/doc 
  5. в качестве endpoint'а для открытия WebSocket'а использовать ws://83.149.26.165/v0/api
  6. медиафайлы приветствий класть на http://83.149.26.165/v0/media
  7. медиафайлы записанных разговоров забирать с http://83.149.26.165/v0/media/rec

Следует помнить, что вызовы к API по сути своей асинхронны - сеть связи представляет собой систему реального времени. Реализация параллелизма вызовов, если она необходима в конкретном приложении, лежит на разработчике и достигается средствами конкретных инструментов разработки (язык программирования, среда исполнения и т.п.).
