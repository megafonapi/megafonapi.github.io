MegaFon - Telecom API : Работа с очередями (в разработке)

Created by Alexey Brykov, last modified by Konstantin Levchin on Nov 14, 2018


Команды работы с очередями позволяют создавать, удалять и обслуживать объекты типа "очередь". 

Очереди используются для организации работы контактных центров, предоставляя возможность управлять состоянием вызова в очереди, информированием пользователя в очереди, передачей вызова выбранному агенту (оператору) путем создания вызова к нему.

Агент - пользователь, принимающий и обрабатывающий вызов из очереди. Максимальный размер очереди и количество одновременно работающих агентов на одной очереди задаются конфигурацией объекта "очередь"

Выбор агента производится по алгоритмам ACD (Automatic Call Distribution) - автоматического распределения вызовов. Поддерживаются алгоритмы:

  • Linear (линейный выбор. Звонки передаются по списку агентов в линейном порядке)
  • Round Robin (перебор по списку. Следующий звонок будет передан следующему агенту из списка) 
  • Weighted (звонки передаются в соотвествии с весом каждого агента)


Команды работы с очередями находятся в разработке.
