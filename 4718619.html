Мегафон - Телеком API : Сообщения об ошибках

Created by Alexey Brykov, last modified by Konstantin Levchin on Nov 14, 2018

Сообщение об ошибке имеет формат:
{
"jsonrpc": "2.0",
"error": {
"code": -32601,
"reason": "520"
},
"id": "1"
}


параметры:

code определяет, в результате работы какой команды пришло сообщение об ошибке. Имеет следующие значения :

Командаcode
MakeCall32001
TerminateCall32002
PlayAnouncement32004
CreateConf32005
DeleteConf32006
AddToConf32007
RemoveFromConf32008




reason принимает следующие значения:

1Соединение завершено (hang up)
0Нормальное завершение
2Сессия не существует
3Сессия заблокирована транзакцией SLOC
9Ошибка в процессе обмена SDP для B2B-тромбонинга
11Не удалось установить B2B-тромбонинг из-за несоответствия конечных точек RTP
232Ошибка при попытке установления RTP-соединения (внутренние ошибки платформы - компонент MLI SNEW/ATCH/RTP1 reply)
520Неопределенная ошибка (Unknown error)
2565Завершение параллельных сессий, связанных через PSKEY
3074Линия занята (SIP 486 Busy Here)
3075Отмена вызова по CNCL или не удалось установить соединение
3076Вызов отклонен
3077Получен ответ SIP 404 Not Found
3078Нет ответа
3080Получен ответ SIP 410 или 480
3081Сервис не доступен (SIP 500) или неверный SDP: SAI SUPR reply INVITE_PROV_RESP/INVITE_2XX невыполнение precondition (e2e, local, remote)
3082Маршрут недоступен
3095Состояние сессии (или SECSESS) входит в списк состояний перечисленных в STATESET FICS
3840Получен REINVITE
