MegaFon - Telecom API : Сообщения об ошибках

Created by Alexey Brykov, last modified by Sergei Naumov on Dec 07, 2018

Сообщение об ошибке имеет формат:

{"jsonrpc": "2.0",
 "error": {
 "code": -32090,
 "reason": "no call session id"
 },
 "id": "1"
}



параметры:
