Мегафон - Телеком API : Работа с файлами

Created by Alexey Brykov, last modified by Konstantin Levchin on Nov 14, 2018

<дописать конкретику>

На сервере БД для владельца MSISDN выделена область home/megafonapi/<MSISDN> с правами доступа на запись владельцу MSISDN.

В эту область сохраняются файлы, связанные с работой приложения (записи разговоров и конференций, проигрываемые сообщения, история  и проч.)

Доступ к области осуществляется через файл-сервер с веб-порталом. Логин - MSISDN, пароль - по дефолту (пока не поменяет пользователь).

Файлы, записываемые системой (параметр record), имеют формат имени файла в виде строки GUID (например, 9fdea735-1068-4736-9cf4-36030602509e.vox, задаются параметром record_id). Это - временные файлы, они будут  удвлены через 1 месяц и должны быть скопированы пользователем до этого времени. Файлы, используемые как промпты и голосовые подсказки (команда PlayAnnouncement, параметр res-filename), имеют формат имени файла в виде текстовой строки "file_name.vox", например "welcome.vox". Эти файлы будут оставаться в системе на все время работы учетной записи ползователя.
