MegaFon - Telecom API : Работа с файлами

Created by Alexey Brykov, last modified by Sergei Naumov on Dec 07, 2018

Пользователь МегаФон.API получает небольшое дисковое пространство для размещения файлов приветствий, (см. команду PlayAnnouncement) и файлов записи разговора. Файлы с записанными разговорами периодически удаляются, поэтому рекомендуается их забирать как можно скорее. Доступ к этим разделам осуществляется по WebDAV. Просмотреть содержимое пространства приветсвий можно прями из командной строки

[serge@yarilo examples]$ curl -X GET --basic --user 79281234567:pass http://192.168.1.67/media/prompts
[
{ "name":"alarm.pcm", "type":"file", "mtime":"Sun, 25 Nov 2018 07:40:26 GMT", "size":123716 },
{ "name":"record.pcm", "type":"file", "mtime":"Fri, 07 Dec 2018 08:17:58 GMT", "size":48160 },
{ "name":"welcome.pcm", "type":"file", "mtime":"Sat, 24 Nov 2018 20:46:28 GMT", "size":9520 },
{ "name":"welcome.wav", "type":"file", "mtime":"Sat, 24 Nov 2018 20:46:58 GMT", "size":19084 }
][serge@yarilo examples]$

или из любого пользовательского кода с использованием соответствующих инструментов. Чтобы положить файл с приветствием, необходимо

curl -X PUT -T login.pcm --user 79281234567:qwerty http://92.168.1.67/media/prompts/login.pcm

А для удаления, необходимо выполнить DELETE:

curl -X DELETE --basic --user 79281234567:qwerty http://192.168.1.67/media/prompts/welcome.wav

Аналогичные операции проделываются и для пространства записанных разговоров. Например для получения на локальный компьютер файла записи вызова надо выполнить команду:

curl -O --user "79281234567:qwerty" http://192.168.1.67/media/records/79281234567_106432.pcm

Для прослушивания файлов pcm можно использовать такие программы как AudaCity (Windows/Max OSX/Linux) или Prompt Studio (Windows). Можно перекодировать, с помощью, например, sox в Linux.

Вероятность, что чуваки будут называть GUID-ом файлы приветствия - конечно есть, но крайне низкая. Поэтому пока можно делать: GUID.parse швыряет exception - значит, файл оставляем)
