Мегафон - Телеком API : Работа с файлами

Created by Alexey Brykov, last modified by Sergei Naumov on Dec 05, 2018

Пользователь МегаФон.API получает небольшое дисковое пространстводля размещения файлов приветствий, (см. команду PlayAnnouncement) и файлов записи разговора. Файлы с записанными разговорами периодически удаляются, поэтому рекомендуается их забирать как можно скорее. Доступ к этим разделам осуществляется по WebDAV. Просмотреть содержимое пространства приветсвий можно прями из командной строки

[serge@yarilo examples]$ curl -X GET --basic --user 79281234567:pass http://192.168.1.67/media/prompts
[
{ "name":"hello.alaw", "type":"file", "mtime":"Sun, 25 Nov 2018 07:40:26 GMT", "size":123716 },
{ "name":"welcome.alaw", "type":"file", "mtime":"Sat, 24 Nov 2018 20:46:28 GMT", "size":9520 },
{ "name":"welcome.wav", "type":"file", "mtime":"Sat, 24 Nov 2018 20:46:58 GMT", "size":19084 }
][serge@yarilo examples]$

или из любого пользовательского кода с использованием соответствующих инструментов. Чтобы положить файл с приветствием, необходимо

curl -X PUT -T login.alaw --user 79281234567:qwerty http://92.168.1.67/media/prompts/login.alaw

А для удаления, необходимо выполнить DELETE:

curl -X DELETE --basic --user 79281234567:qwerty http://192.168.1.67/media/prompts/welcome.wav

Аналогичные операции проделываются и для пространства записанных разговоров
