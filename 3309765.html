MegaFon - Telecom API : Отправка SMS (в разработке)

Created by Alexey Brykov, last modified by Konstantin Levchin on Nov 14, 2018


Команда позволяет отправлять мобильному абоненту сети связи общего пользования короткое сообщение (SMS) с использованием протокола SMPP. 

Передача сообщения производится согласно стандартов 3GPP TS 23.040

MSISDN получателя может быть:

  • местный мобильный номер абонента
  • междугородний мобильный номер абонента
